泵：步进电机中断，抢占优先级为0
1、1000ml 3000ml泵，步进分频系数8，电子齿轮比分子（F 0F）=625，电子齿轮比分母（F 10）=100；

泵流速校正步骤：
1、在PC软件里点击【进入流速校正】，进入流速校正模式；
2、在空载状态下，发送待测试的流速点（理论流速），测实际流速；
3、填充流速校正表；
4、估算实际最大流速对应的理论流速点，最好大于或等于实际最大流速，填充最后一个校正数据；
5、发送流速校正表到主板，退出流速校正模式；
6、检验校正后的流速是否达到指标，如果不满足，则重新校正不满足的点，或者在该点附近添加更多校正点；
7、加载测试，发送加载流速系数到主板，该系数=校正后的空载流速/校正并加载后的实际流速；
